Detailed Description

Module to play motion files.

This class implements a Motion Player for Motion files

Author:
Stefan Kaufmann A Motion File is a simple text file with Motion Information. You can add comments in a Motion file by "//". A Motion file has four commands:

  • NAME:
  • UPDATE:
  • TIMES:
  • COMMANDS:

Each command ends with an ";". There can only be one command per line. An example of an Motion file can be: 

 NAME:        "Head"; 
 UPDATE:      "ClearAll"; 
 TIMES:       1000    2000    3000    4000; 
 COMMANDS:    0.5     0.8     0.0     -0.5; // HeadYaw  
              0.2     0.5     -0.1    -0.5; // HeadPitch
